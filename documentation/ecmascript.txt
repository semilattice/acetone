Usage

Build the runtime with the following command:

    nix-build -A runtime.ecmascript

Now you can invoke the linker with the output of Acetone, like so:

    result/bin/link unit1.js unit2.js unit3.js ...

Internals

AL is the prefix for locals.
AG is the name of the object in which globals are stored.
AR is the name of the object in which runtime functions are stored.
